# User App

Двухстраничное приложение на React и TypeScript для управления списком пользователей. Приложение позволяет просматривать список пользователей, редактировать их данные, а также управлять состоянием карточек (архивация, скрытие).

Данные загружаются с внешнего API: [JSONPlaceholder](https://jsonplaceholder.typicode.com/). Также реализованы состояния загрузки данных и обработки ошибок.

### Основной функционал

**Главная страница:**

- Отображение списка пользователей с ключевой информацией (username, город, компания, аватар)
- Действия с карточками: редактирование, архивирование, скрытие, восстановление из архива
- Разделение пользователей на активных и архивных

**Страница редактирования:**

- Форма редактирования данных пользователя
- Валидация пользовательского ввода
- Отображение уведомления об успешном сохранении (модальное окно с автозакрытием)

**Техническая реализация:**

1. Управление состоянием - Zustand
2. Работа с серверными данными - TanStack Query
3. Формы и валидация - React Hook Form + Zod
4. Маршрутизация - React Router
5. Стилизация - SCSS
